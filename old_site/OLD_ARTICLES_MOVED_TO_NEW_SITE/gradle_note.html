В проекте описано все, что нужно для сборки и deploy. Более детально "Организация собственного CI/CD".

Примечания:
  • Шпаргалка по Gradle
  • Вывод gradle tasks ./gradlew tasks
  • Не найден plugin при сборке gradle
    Добавить в начало: 
    buildscript {
  repositories {
    mavenCentral()
 }
}
    Это позволит искать плагин в центральном репозитории maven. Корневая секция (repositories {}) предназначена для dependencies , но не для plugins (buildscript{repositories{}}) 
    repositories {
  mavenCentral()
  }
}
