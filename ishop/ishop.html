Языки программирования: Java, Python, JS
Системная часть: Tomcat, PostgreSQL, ActiveMQ, OpenVPN, Camel
Интерфейсная часть: Eclipse RCP (сама по себе интересная плагинная, мультиплатформенная система для разработки desktop приложений, еще и OSGi), React, позже Vue.

Кратко функции системы:

  • Контроль остатков магазинов, средств предприятия, анализ остатков поставщиков, заказ (ручной в случае если постащик не имел системы заказа и автоматом через системы поставщиков).
  • Расценка новых товаров, проведение торговых акций и переоценка товаров.
  • Рабочее место продавца, директора магазина, менеджеров по закупкам/продажам, собственников, бухгалтеров. Сделаны с использованием Ecpipse RCP и web интерфейсов.
  • Разработка обязанностей сотрудников, средства и сроки выполнения задач в должностных инструкциях.
  • Выгрузка отчетов для бухгалтерии в формате 1С и Excel.
  • По результатам продаж, расчет ЗП менеджеров по закупкам/продажам, управляющих магазинов, продавцов.
  • ABC анализ, категоризация товаров по товарным группам, продаваемости, новинки.
Интернет магазин (на базе React). Тут же подсистема доставки.

Разработана специальная система подготовки описаний товаров с парсингом характеристик с сайтов производителей и формирования ценников. Справочник товаров ~1 млн.
