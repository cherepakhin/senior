Easycode

https://easycode.ru/ idea plugin есть @Service @Slf4j @AllArgsConstructor public class AnimalsServiceImpl implements AnimalsService { @Override public List getAll() { Animal animal = new Animal(); return List.of(animal); } } выделить и в контекстном меню выбрать из вариантов: "What does this do?" "Write Unit tests" "Refactor code" "Custom Prompt" в панели EasyCode будет ответ.

Пример ответа для prompt "Write Unit tests":
@RunWith(MockitoJUnitRunner.class)
public class AnimalsServiceImplTest {

    @Mock
    private AnimalRepository animalRepository;

    @InjectMocks
    private AnimalsServiceImpl animalsService;

    @Test
    public void testGetAll() {
        // Arrange
        List<> animals = new ArrayList<>();
        animals.add(new Animal());
        Mockito.when(animalRepository.findAll()).thenReturn(animals);

        // Act
        List<> result = animalsService.getAll();

        // Assert
        Mockito.verify(animalRepository, Mockito.times(1)).findAll();
        Assert.assertEquals(animals, result);
    }
}
