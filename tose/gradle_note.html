Не найден plugin при сборке gradle Добавь в начало 

buildscript {
  repositories {
    mavenCentral()
  }
}



repositories {
  mavenCentral()
  }
}
Это позволит искать твой плагин в центральном репозитории maven. Обрати внимание, что корневая секция 

repositories {
  mavenCentral()
  }
}
предназначена для dependencies, но не для plugins
