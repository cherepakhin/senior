Примечания:
  • Шпаргалка по Gradle
  • Вывод gradle tasks ./gradlew tasks
  • Не найден plugin при сборке gradle
    Добавь в начало 
    buildscript {
  repositories {
    mavenCentral()
  }
}
    Это позволит искать твой плагин в центральном репозитории maven. Обрати внимание, что корневая секция (repositories {}) предназначена для dependencies , но не для plugins (buildscript{repositories{}}) 
    repositories {
  mavenCentral()
  }
}
