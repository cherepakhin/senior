Autocompete code с помощью MCP Server и Visual Studo Code. Пример настройки применения MCP сервиса github для генерации подсказок. В корне проекта должен быть файл .vscode/.mcp.json со следуюшим содержанием: 
{
    "servers": {
      "MCP Docs": {
        "type": "sse",
        "url": "https://gitmcp.io/cherepakhin/project"
      }
    }
  }
Где: https://gitmcp.io/cherepakhin/project - это ссылка на репозиторий с проектом, по которому будет производиться поиск подсказок.
