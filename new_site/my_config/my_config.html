Ссылка на репозиторий: FileUploadExample В проекте используется ЧАСТНЫЙ параметр конфигурации myconfig:catalogForSave из application.yml: 
myconfig:
  catalogForSave: ${user.home}/temp/1
Использование в Spring bean:
@Controller
public class UploadController {
    ...
    @Value("${myconfig.catalogForSave}")
    String catalogForSave;

    @PostConstruct
    void init() {
        logger.info("=====================");
        logger.info("Catalog For Save location= '{}'", catalogForSave);
        logger.info("=====================");
    }
    ...
}
Для контроля создан метод с аннотацией @PostConstruct.
