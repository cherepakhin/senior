Kafka Drop

https://github.com/obsidiandynamics/kafdrop.
Статья про Kafka Drop из "Русские Блоги": https://russianblogs.com/article/92143148179/
Расположена в v.perm.ru:/home/vasi/tools/kafka_drop/. Нужна JAVA 17. Запуск UI Kafdrop (vasi@v ~/tools/kafka_drop/run.sh): 
vasi@v ~/tools/kafka_drop$ /usr/lib/jvm/java-17-openjdk-amd64/bin/java -jar target/kafdrop-4.0.2-SNAPSHOT.jar --kafka.brokerConnect=192.168.1.20:9092
Доступ из браузера: http://192.168.1.20:9000 Основной экран:

Kafka UI

https://github.com/provectus/kafka-ui
Запуск Kafka UI: 
$ /usr/lib/jvm/java-1.17.0-openjdk-amd64/bin/java --add-opens java.rmi/javax.rmi.ssl=ALL-UNNAMED -jar kafka-ui-api-v0.7.1.jar
или 
vasi@v ~/tools/kafka_ui $./start.sh

Доступ из браузера: http://192.168.1.20:8960/ui/clusters/local/all-topics?perPage=25
