Скачивание с youtube, smotrim.ru и т.п. В чем проблема?
Связь, которую предоставляет мой провайдер, бывает с проблемами. Утром почти никакая - паузы, видео недоступно и прочая шняга. Хочется скачать видео и нормально его просмотреть. Нашлась отличная утилита скачивания видео https://github.com/yt-dlp/yt-dlp. Есть версии для Windows, Linux, Mac.
Установка в linux: 
sudo apt install ffmpeg
sudo apt install yt-dlp -y
Использование: 
$ yt-dlp https://smotrim.ru/video/123456
