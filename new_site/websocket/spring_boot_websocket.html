WebSocket (веб-сокет) — сетевой протокол передачи данных, который обеспечивает постоянное двустороннее (full-duplex) соединение между клиентом и сервером через одно TCP-соединение. Главная особенность — поддержка актуальности соединения без новых запросов от клиента.

Работа с WebSocket со Spring Boot описана Readme.md проекта https://github.com/cherepakhin/messaging-stomp-websocket. Задача проекта в передаче сообщений через веб-сокеты между браузерами клиентов, находящихся на разных машинах.
