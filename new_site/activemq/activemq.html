Файлы настройки ActiveMQ находятся в папке v:/home/vasi/po/activemq/conf.
Копии файлов в https://github.com/cherepakhin/senior/tree/main/new_site/activemq/conf
activemq.service
[Unit]
Description=activemq
Wants=network-online.target
After=network-online.target
[Service]
Type=simple
User=vasi
Group=vasi
ExecReload=/bin/kill -HUP
ExecStart=/home/vasi/po/activemq/bin/activemq start
SyslogIdentifier=activemq
StandardOutput=file:/var/log/activemq.log
Restart=always
[Install]
WantedBy=multi-user.target
Запуск сервиса:
sudo systemctl start activemq.service
Остановка сервиса:
sudo systemctl start activemq.service
