Селектор по id
https://htmlacademy.ru/blog/css/id-selector

<div id="uniqueElement">Этот элемент уникален.</div>

#uniqueElement {
  color: green;
  font-size: 20px;
}

Идентификаторы в HTML должны быть уникальными в пределах документа
Высокий приоритет: Стили, примененные через селектор по id, имеют более высокий приоритет, чем стили, примененные через классы и теги
========================
Список селекторов в CSS представляет собой перечень селекторов, разделенных запятыми
h1, h2, p, li {
  color: #333;
  font-family: 'Arial', sans-serif;
}
========================
Смежный селектор "+"
Смежный селектор в CSS, обозначаемый знаком плюс (+), позволяет выбирать элемент, который является непосредственным соседом другого элемента на одном и том же уровне иерархии

h2 + p {
  color: #757575;
  margin-top: 0;
}
В данном случае к абзацу (<p>), непосредственно следующему за заголовком второго уровня (<h2>), применяются определенные стили: цвет текста изменится на серый, а верхний отступ будет устранен.
========================
Селектор тега

p {
  font-family: "Helvetica", sans-serif;
  line-height: 1.6;
  color: #333;
}

 может быть переопределена дополнительными селекторами, например, селекторами класса или идентификатора.
========================
Универсальный селектор *

* {
  margin: 0;
  padding: 0;
  font-family: Arial, sans-serif;
}

используется для применения стилей ко всем элементам на веб-странице
