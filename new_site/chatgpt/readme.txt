Коротко об ИИ.
https://habr.com/ru/articles/947216/#1
Эмбеддинг — это массив чисел, который получается преобразованием какого‑то объекта (например текста или картинки).

Пример эмбединга  - описание точки с помощью координат x,y.

В реальности, работая с эмбеддингами, вектора могут иметь размерность и 512, 1024 и более.

Нужно узнать насколько похожи два вектора. Есть разные варианты:
1. Евклидово расстояние
2. Манхэттенское расстояние
3. Косинусная близость
4. Косинусное расстояние

 Косяки ИИ
 1. Использует не точные данные, пытается угодить клиенту (вспомни опыты с нейросетью)
 2. Мультиплицирует наши ошибки
 3. Информационный пузырь (хайп и т.п.)
 4. Искушение переложить ответственность (тут же где-то лень)
