Статья про Kafka Drop из "Русские Блоги": https://russianblogs.com/article/92143148179/ (расположена в v.perm.ru:/home/vasi/tools/kafka_drop/. Нужна JAVA 17.). Запуск UI Kafdrop: 
vasi@v ~/tools/kafka_drop$ /usr/lib/jvm/java-17-openjdk-amd64/bin/java -jar target/kafdrop-4.0.2-SNAPSHOT.jar --kafka.brokerConnect=192.168.1.20:9092
(vasi@v ~/tools/kafka_drop/run.sh).
